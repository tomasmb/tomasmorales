# tomasmorales.dev

Personal portfolio website built with Next.js 15.

## Features

- Multilingual (English/Spanish)
- Responsive design
- SEO optimized
- Deployed on Vercel

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 15 + React 19
- TypeScript
- Tailwind CSS 4
- Framer Motion
- next-intl
